# 今天吃什么

帮你决定「这顿吃什么」的 Web 小应用。定位附近餐厅，结合心情和偏好，由 AI 从真实店铺里挑出最值得去的几家。

## 它能做什么

- **附近探店**：基于当前位置，搜索 2km 内的餐饮店铺（高德 POI，最多约 150 家）
- **快速选偏好**：心情、口味、菜系、预算，支持自定义输入和其他要求
- **AI 推荐 Top 5**：从附近真实餐厅中推荐 5 家，并给出简短理由
- **换一批**：不换选项，排除已看过的店，重新生成推荐
- **一键导航**：点击「去这家！」跳转地图路线规划，餐厅自动设为终点
- **偏好记忆**：每次「去这家」会记录你的选择，用于后续推荐
- **饮食日历**：记录每日三餐吃了什么，按月查看饮食轨迹

## 使用流程

1. 打开应用，允许浏览器定位（或使用网络定位）
2. 选择心情、口味、想吃啥、预算等
3. 点击「帮我决定！」
4. 查看 Top 5 推荐，满意就「去这家！」，不满意就「换一批」
5. 在「饮食日历」查看历史记录

## 技术栈

| 模块 | 技术 |
|------|------|
| 前端 | React + Vite + Tailwind CSS |
| 后端 | Node.js + Express |
| 地图 / POI | 高德地图 Web 服务 API |
| AI 推荐 | OpenAI 兼容接口（如 DeepSeek） |
| 数据存储 | 浏览器 localStorage（饮食记录、偏好） |

## 本地运行

```bash
cd jintian-chi-shenme
cp .env.example .env   # 填入高德 Key 和 AI Key
npm install
npm run dev
```

浏览器访问 **http://localhost:5173**

### 环境变量

| 变量 | 说明 |
|------|------|
| `AMAP_KEY` | 高德 Web 服务 Key |
| `AI_API_KEY` | 大模型 API Key |
| `AI_BASE_URL` | API 地址，默认 DeepSeek |
| `MOCK_MODE` | `false` 使用真实数据 |

修改 `.env` 后需重启 `npm run dev`。

## 说明

- 定位为浏览器 GPS，室内或 Mac 上可能较慢，可用「网络定位」兜底
- 饮食记录仅存本地，换设备或清缓存会丢失
- 后续可扩展为微信小程序，后端逻辑可复用
